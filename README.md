# The "API Quiz App" Debugging Challenge

Welcome! Your job is to fix a buggy and incomplete Quiz Application.
A previous developer left this project partially implemented. There are multiple UI issues, logic errors, and missing features.

Your mission is to:

1. Identify and fix the bugs.
2. Add the missing functionalities.

---

# HOW TO START

1. Open the project folder.
2. The app contains:

    `index.html`
    `style.css`
    `script.js`

3. Open `index.html` in any browser to run the quiz.
4. Internet connection is required because the quiz uses an external API.

---

# WHAT YOU HAVE (EXISTING CODE)

###  Basic Quiz Logic

* Fetches questions from Open Trivia DB API
* Displays one question at a time
* Shows correct/incorrect feedback
* Tracks score
